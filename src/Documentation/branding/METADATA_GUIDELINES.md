# Metadata Guidelines - Localized SEO Strategy

## Overview

This document defines **where and how** to apply localized keywords versus maintaining global brand consistency across all Nom Stories channels. Following these guidelines ensures maximum discoverability while preserving brand identity.

---

## Core Principle

**Global Brand + Local Discovery**

- **Channel-level**: Keep global brand consistency (Nom + format)
- **Video-level**: Maximize local SEO and discoverability
- **Content-level**: Full cultural adaptation

---

## Metadata Hierarchy

### Channel Level (Global Brand Focus)

#### Channel Name ✅ Global
**Format:** `Nom [Content Type] [LANG]`

**Rules:**
- Always include "Nom" brand name
- Keep English descriptors: "Stories", "Scary Stories"
- Add 2-letter language code
- No localization

**Examples:**
```
✅ Nom Stories (EN)
✅ Nom Scary Stories ES
✅ Nom Scary Stories CZ
❌ Nom Strašidelné Příběhy CZ (localized descriptor)
❌ Historias de Nom ES (reversed order)
```

#### Channel Handle ✅ Global
**Format:** `@Nom[ContentType][Lang]`

**Rules:**
- CamelCase, no spaces
- English keywords only
- Language code at end

**Examples:**
```
✅ @NomStoriesEN
✅ @NomScaryStoriesES
✅ @NomScaryStoriesCZ
❌ @NomPribehyCZ
❌ @nom_scary_stories_es
```

#### Channel Description 🌐 Bilingual
**Format:** English summary + Native language description

**Structure:**
```
[English - 1 sentence brand intro]
[Empty line]
[Native language - 2-3 sentences description]
[Native language - content promise]
[Keywords in native language]
```

**Example (Spanish):**
```
Nom Scary Stories brings you the best horror content.

Historias de terror reales que te mantendrán despierto por la noche. 
Sube nuevo contenido cada semana. Suscríbete para más historias escalofriantes.

Palabras clave: historias de terror, miedo, horror, cuentos de miedo, 
relatos escalofriantes, videos de terror
```

**Example (Czech):**
```
Nom Scary Stories brings you the best horror content.

Skutečné strašidelné příběhy, které vás udrží bdělé celou noc. 
Nový obsah každý týden. Přihlaste se k odběru pro více děsivých příběhů.

Klíčová slova: strašidelné příběhy, horor, děsivé příběhy, 
scary stories česky, hororové příběhy
```

#### Channel Keywords 🌐 Mixed
**Distribution:** 50% English / 50% Native Language

**Strategy:**
- Include "Nom" and "Nom Stories"
- Mix English high-volume terms
- Add native language equivalents
- Include genre-specific terms

**Example (German):**
```
English: Nom, Nom Stories, scary stories, horror, creepy, short horror
German: Gruselgeschichten, Horror, gruselig, kurze Horrorgeschichten, 
        gruselige Geschichten, Horrorgeschichten Deutsch
```

#### Channel Art (Banner) ✅ Global + Language Indicator
**Elements:**
- Nom logo (consistent)
- "Scary Stories" in English
- Language indicator: small flag or "[CZ]" badge
- Visual style consistent with brand
- High contrast, mobile-optimized

## Localization Quality Checklist

### Before Publishing

- [ ] **Title**: Native language, keywords optimized, engaging
- [ ] **Description**: Native language, first 3 lines strong, CTA included
- [ ] **Tags**: 70% native / 30% English, all relevant
- [ ] **Hashtags**: Native language, mix trending + niche
- [ ] **Thumbnail**: Native text, readable on mobile, emotion clear
- [ ] **Voiceover**: Native speaker quality, clear pronunciation
- [ ] **Captions**: Synced perfectly, native language, readable
- [ ] **Content**: Culturally adapted, not just translated
- [ ] **Cultural sensitivity**: Reviewed for appropriateness
- [ ] **Brand consistency**: Channel name, logo, handle correct

### Quality Standards

**Tier 1 Markets (US, ES, DE):**
- Native speaker voiceover mandatory
- Professional translation/localization
- Cultural consultant review
- A/B tested titles and thumbnails

**Tier 2 Markets (CZ, PL):**
- Native quality TTS or native speaker
- Localization (not just translation)
- Basic cultural adaptation
- Standard thumbnail templates

**Tier 3 Markets (Future):**
- High-quality TTS acceptable
- Professional translation
- Standard adaptations
- Template-based thumbnails

## Keyword Research Process

### 1. Identify High-Volume Terms

**Tools:**
- Google Keyword Planner (by language/region)
- YouTube Search Autocomplete
- TubeBuddy / VidIQ (language-specific)
- Google Trends (by country)

**Process:**
```
1. Enter seed keyword in native language (e.g., "historias de terror")
2. Export related keywords with search volume
3. Filter for relevant, high-volume terms
4. Prioritize terms with low competition
5. Create keyword database per language
```

### 2. Analyze Competitors

**Research:**
- Find top channels in native language
- Analyze their most popular videos
- Extract title patterns and keywords
- Note hashtag strategies
- Document thumbnail styles

**Tools:**
- Social Blade (channel stats)
- YouTube search filters (by language)
- Manual competitive analysis
- VidIQ competitor tracking

### 3. Test and Iterate

**Process:**
```
1. Create baseline video with standard keywords
2. Measure CTR, views, engagement
3. A/B test title variations
4. Update keyword strategy based on data
5. Repeat monthly
```

## Translation vs. Localization

### ❌ Translation (Don't Do This)

**Word-for-word translation:**
```
English: "I was at Starbucks studying for the SAT"
Spanish: "Estaba en Starbucks estudiando para el SAT"
```
*Problem: Starbucks may not be common, SAT doesn't exist in Spanish-speaking countries*

### ✅ Localization (Do This)

**Culturally adapted:**
```
English: "I was at Starbucks studying for the SAT"
Spanish: "Estaba en el café estudiando para el examen de admisión"
```
*Better: Generic café, local exam system*

### Localization Checklist

**Always Adapt:**
- [ ] Brand names (if not globally known)
- [ ] Educational systems (SAT → local equivalent)
- [ ] Measurement units (Fahrenheit → Celsius)
- [ ] Cultural references (American football → local sport)
- [ ] Idioms and expressions
- [ ] Names (John → Juan / Jan / Johann)
- [ ] Locations (New York → local city)
- [ ] Food and drink references
- [ ] Holiday and seasonal references
- [ ] Social norms and behaviors
